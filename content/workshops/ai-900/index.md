## Microsoft Azure AI Fundamentals Certification (AI-900)

The Microsoft Azure AI Fundamentals certification is designed for those seeking to demonstrate foundational knowledge of AI and its practical implementation through Microsoft Azure's services. It is suitable for a broad audience including IT professionals, business decision-makers, data engineers, and students interested in understanding the potential of AI.

### Exam Objectives

The AI-900 exam assesses understanding in the following domains:

1. **AI Workloads and Considerations (15-20%)**

   - Identify features of common AI workloads.
   - Identify guiding principles for responsible AI.
2. **Fundamental Principles of Machine Learning on Azure (30-35%)**

   - Identify common machine learning types.
   - Describe core machine learning concepts.
   - Identify features of computer vision workloads.
3. **Features of Computer Vision Workloads on Azure (15-20%)**

   - Identify features of computer vision workloads.
   - Identify Azure tools and services for computer vision.
4. **Features of Natural Language Processing (NLP) Workloads on Azure (15-20%)**

   - Identify features and challenges of NLP workloads.
   - Identify Azure tools and services for NLP.
5. **Features of Conversational AI Workloads on Azure (15-20%)**

   - Identify features and components of conversational AI solutions.
   - Identify Azure services for conversational AI.

### Key Skills Measured

The AI-900 certification measures several skills, including:

- Describing AI concepts and workloads.
- Identifying Azure AI services related to specific workloads (e.g., computer vision, NLP).
- Understanding machine learning models and workflows.
- Implementing responsible AI practices.

### Benefits of AI-900 Certification

- **Career Advancement:** Demonstrates foundational knowledge to employers.
- **Business Impact:** Enables stakeholders to make informed decisions on AI strategy.
- **Solid Foundation:** Prepares candidates for advanced certifications like the Azure AI Engineer Associate.

## Resources for Exam Preparation

Microsoft provides various resources to help candidates prepare for the AI-900 certification exam.

### Learning Paths and Courses

 **Microsoft Learn Training (AI-900T00):**
   A free, comprehensive training course available on Microsoft's Learn platform. The course includes modules like:

- **Introduction to AI Concepts:** Overview of machine learning and AI workloads.
- **Responsible AI Principles:** How to implement ethical and responsible AI.
- **Machine Learning Fundamentals:** Core concepts, supervised and unsupervised learning.
- **Computer Vision on Azure:** Leveraging Azure Cognitive Services for computer vision workloads.
- **Natural Language Processing (NLP):** Azure tools for text analysis and language understanding.
- **Conversational AI Solutions:** Implementing chatbots with Azure Bot Services.

   Link: [Microsoft Learn AI-900T00 Course](https://learn.microsoft.com/en-us/training/courses/ai-900t00)

### Certification Study Guide and Practice Assessments

**Study Guide:**
   A downloadable PDF study guide is available on Microsoft's website, providing a detailed outline of exam objectives and study materials.

   Link: [AI-900 Certification Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-900)

**Practice Assessments:**
   Microsoft offers free practice assessments to help candidates familiarize themselves with the exam format and question types.

   Link: [AI-900 Practice Assessment](https://learn.microsoft.com/en-gb/credentials/certifications/azure-ai-fundamentals/?practice-assessment-type=certification)
